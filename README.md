# Advocacia Prestígio - Landing Page

Página institucional para escritório de advocacia com design responsivo.

## ⚙️ Tecnologias
- HTML5, CSS3, JavaScript puro

## 📱 Funcionalidades
- Menu fixo com rolagem suave
- Design responsivo (mobile/desktop)
- 8 cards de serviços
- Seções: Hero, Atuação, Diferenciais, Sobre, Contato
- Botão que direciona para página de contato

## 🚀 Como usar
1. Salve `index.html` e `contato.html` na mesma pasta
2. Abra `index.html` no navegador

## 📁 Arquivos
- `index.html` - Página principal
- `contato.html` - Página de contato

## 🎨 Cores
- Azul Escuro: `#0a1128`
- Dourado: `#d4af37`
